Params

Params stores `x/vault` parameters.

FieldTypeLabelDescription
layers uint32

The number of layers of orders a vault places. For example if `layers=2`, a vault places 2 asks and 2 bids.

spread_min_ppm uint32

The minimum base spread when a vault quotes around reservation price.

spread_buffer_ppm uint32

The buffer amount to add to min_price_change_ppm to arrive at `spread` according to formula: `spread = max(spread_min_ppm, min_price_change_ppm + spread_buffer_ppm)`.

skew_factor_ppm uint32

The factor that determines how aggressive a vault skews its orders.

order_size_pct_ppm uint32

The percentage of vault equity that each order is sized at.

order_expiration_seconds uint32

The duration that a vault's orders are valid for.
